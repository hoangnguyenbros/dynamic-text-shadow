# FunTrackerBlod

A small web project that displays an animated "FUN" splash screen and adds a dynamic mouse-tracking shadow effect to the text.

## Features

- Full-screen red background
- Large white text with the word `FUN`
- Dynamic text shadow that follows mouse movement

## How to run

1. Open `FunTrackerBlod/index.html` in a web browser.
2. Move your mouse around the page to see the text shadow effect.

## Notes

- You can customize the text, colors, and shadow behavior by editing the HTML and style section.
